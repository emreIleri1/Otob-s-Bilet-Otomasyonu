# [ Otobüs Bilet Otomasyonu ]

Bu proje, **Veritabanı Yönetim Sistemleri** dersi kapsamında geliştirilmiş, veri odaklı bir web uygulamasıdır. Projenin temel amacı, karmaşık veri ilişkilerini yöneten ve kullanıcı dostu bir arayüz sunan tam kapsamlı bir sistem oluşturmaktır.

## 🚀 Öne Çıkan Özellikler
* **Dinamik Veri Yönetimi:** CRUD (Oluşturma, Okuma, Güncelleme, Silme) işlemlerinin tamamı desteklenmektedir.
* **Modern Web Mimari:** Geliştirme sürecinde **Antigravity** framework'ü kullanılarak performanslı bir yapı kurulmuştur.
* **İlişkisel Veritabanı:** Normalize edilmiş tablolar ve optimize edilmiş SQL sorguları ile veri bütünlüğü sağlanmıştır.
* **Full-Stack Yaklaşım:** Uygulama hem ön yüz hem de arka yüz süreçlerini kapsamaktadır.

## 🛠️ Kullanılan Teknolojiler
* **Framework:** Antigravity
* **Veritabanı:** [ MySQL]
* **Programlama Dili:** [PHP]
* **Konteynerleştirme:** Docker (Proje izolasyonu ve kolay kurulum için)
* **Proje Yönetimi:** Jira (Ekip içi görev dağılımı ve süreç takibi için)
